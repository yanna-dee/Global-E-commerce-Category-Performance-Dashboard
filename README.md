# Global E-commerce Category Performance Dashboard (Tableau)
## Overview
This project showcases an interactive Tableau dashboard analyzing global e-commerce performance. It highlights key business metrics such as revenue, total orders, and average order value (AOV), along with category-level and geographic insights.

The goal is to uncover trends and patterns that can support executive-level decision-making in e-commerce.

## Key Features
- KPI cards for Revenue, Orders, and AOV with month-over-month growth
- Monthly trend line chart for performance tracking
- Category-wise revenue breakdown
- Global map showing country-level sales distribution
- Interactive filtering by month
